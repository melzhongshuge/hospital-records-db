# Hospital Patient Records Database
A MySQL database for managing hospital patient records and medical history 
using relational schema design and SQL queries.

## Tech Stack
MySQL · MySQL Workbench

## Features
- Patient and medical records schema with foreign key constraints
- JOIN queries (INNER, LEFT, RIGHT)
- Filters using LIKE, BETWEEN, DISTINCT, AND/OR operators

## How to Run
1. Open MySQL Workbench
2. Run hospital_records.sql
